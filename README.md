## 安装
```bash
    npm install -s @icanvas/core
```
## 调用
### cjs
```html
    <script src="@icanvas/core/dist/icanvas.web.js">//window.ICanvas
```
### 微信
```javascript
   const ICanvas = require('@icanvas/core/dist/icanvas.wxgame.js');
```
### umd/es6
es6调用完整支持Tree Shake
```javascript
    const ICanvas = require('@icanvas/core')
```
